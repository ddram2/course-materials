# Session 1. 시연용 샘플 Vault 구조

> 이 파일은 워크숍에서 Obsidian의 폴더 구조와 노트 연결을 시연하기 위한 안내서입니다.

---

## 추천 폴더 구조 (교수님 Vault 참고)

```
내연구노트/
├── 00_Inbox/           ← 빠른 메모, 임시 노트
├── 01_Projects/        ← 진행 중인 연구 프로젝트
│   ├── 석사논문/
│   └── 수업과제/
├── 02_Areas/           ← 지속적으로 관리하는 영역
│   ├── 통계방법론/
│   ├── 환경보건/
│   └── 프로그래밍/
├── 03_Resources/       ← 논문 메모, 참고 자료
│   ├── 논문노트/
│   └── 책노트/
├── 04_Archive/         ← 완료된 프로젝트
└── Templates/          ← 노트 템플릿
```

> **핵심**: 폴더는 단순하게, 연결은 `[[링크]]`로!  
> 폴더를 너무 세분화하지 말고, 노트 간 링크로 관계를 만드세요.

---

## 시연 시나리오

### 1. 논문 노트 만들기
→ `03_Resources/논문노트/` 에 `Kim2024_PFAS_건강영향.md` 생성

### 2. 프로젝트 노트에서 논문 노트 연결
→ `01_Projects/석사논문/` 에서 `[[Kim2024_PFAS_건강영향]]` 링크

### 3. 그래프 뷰로 연결 시각화
→ Obsidian 그래프 뷰 열어서 연결된 모습 보여주기
