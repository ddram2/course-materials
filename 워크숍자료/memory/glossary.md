# Glossary

환경보건 연구 및 AI 활용 관련 용어 정리.

## 연구 분야 용어
| Term | Meaning | Context |
|------|---------|---------|
| PFAS | Per- and polyfluoroalkyl substances | 과불화합물, 주요 연구 대상 |
| PBPK | Physiologically-based pharmacokinetic modeling | 생리학적 약동학 모델 |
| EDCs | Endocrine disrupting chemicals | 내분비교란물질 |
| Exposome | 익스포좀 | 전생애 환경노출 총체 |
| KoNEHS | 국민환경보건기초조사 | 한국 인체 바이오모니터링 조사 |
| TSH | Thyroid-stimulating hormone | 갑상선자극호르몬 |
| OR | Odds Ratio | 교차비 |
| CI | Confidence Interval | 신뢰구간 |
