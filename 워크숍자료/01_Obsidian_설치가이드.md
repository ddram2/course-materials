# Obsidian 설치 및 시작 가이드

> 워크숍 사전 배포용 | 설치 후 참석해 주세요

---

## 1단계: 다운로드 및 설치

1. [https://obsidian.md](https://obsidian.md) 접속
2. **Download** 클릭 → 본인 OS (Windows / Mac) 선택
3. 설치 파일 실행 → 기본 설정으로 설치 완료

---

## 2단계: Vault(볼트) 만들기

1. Obsidian 실행
2. **Create new vault** 선택
3. Vault 이름 입력 (예: `내연구노트`)
4. 저장 위치 선택 → **Create** 클릭

> **Vault란?** Obsidian에서 관리하는 하나의 폴더입니다. 이 폴더 안의 모든 `.md` 파일이 연결됩니다.

---

## 3단계: 첫 노트 만들기

1. 왼쪽 상단 **새 노트 만들기** 아이콘 클릭 (또는 `Ctrl/Cmd + N`)
2. 제목 입력 (예: `테스트 노트`)
3. 본문에 아무 내용이나 적어보기:

```markdown
# 나의 첫 노트

오늘 Obsidian을 설치했다.
앞으로 **연구 메모**를 여기에 정리할 예정이다.

관련 노트: [[]]
```

---

## 4단계: 추천 설정

| 설정 항목 | 경로 | 추천값 |
|---|---|---|
| 기본 저장 위치 | 설정 → 파일 및 링크 | Vault 내 하위 폴더 지정 |
| 맞춤법 검사 | 설정 → 에디터 | 끄기 (한글 오탐 많음) |
| 커뮤니티 플러그인 | 설정 → 커뮤니티 플러그인 | 활성화 |

---

## 참고: 유용한 단축키

| 단축키 (Mac/Win) | 기능 |
|---|---|
| `Cmd/Ctrl + N` | 새 노트 |
| `Cmd/Ctrl + O` | 빠른 열기 (노트 검색) |
| `Cmd/Ctrl + E` | 편집/미리보기 전환 |
| `Cmd/Ctrl + [[` | 내부 링크 만들기 |
| `Cmd/Ctrl + P` | 명령어 팔레트 |
| `Cmd/Ctrl + Shift + F` | Vault 전체 검색 |
